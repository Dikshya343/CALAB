
##  TOPIC :  VHDL Code for Realizing Logic Gates



##  OBJECTIVE:
• To write VHDL code for basic logic gates: AND, OR, NOT, NAND, NOR, XOR, and
XNOR.
• To simulate each gate and verify its truth table using GTKWave.



##  THEORY:
Logic gates are the fundamental building blocks of all digital circuits. Each gate performs a
basic Boolean operation on one or more binary inputs to produce a single binary output.

Gate VHDL Operator Boolean Expression
AND and Y = A · B
OR or Y = A + B
NOT not Y = A
NAND nand Y = A · B
NOR nor Y = A + B
XOR xor Y = A ⊕ B
XNOR xnor Y = A ⊕ B




## DISCUSSION:
The VHDL code for realizing logic gates was implemented and simulated successfully. 
Different logic gates such as AND, OR, NOT, NAND, NOR, XOR, and XNOR were designed using VHDL language. 
The design.vhd file contained the logic gate implementation, while the testbench.vhd file was used to apply input 
combinations and verify the outputs. The simulation results matched the expected truth tables of the respective logic gates,
demonstrating the correct functionality of the design. This experiment helped in understanding the basic structure of VHDL coding, 
entity–architecture modeling, and simulation using a testbench.

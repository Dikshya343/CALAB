## TOPIC:  VHDL Code for Combinational  Circuits (MUX and DEMUX):


## Objective:
• To design and simulate a 4-to-1 Multiplexer (MUX) in VHDL.
• To design and simulate a 1-to-4 Demultiplexer (DEMUX) in VHDL.


## Theory:

## Multiplexer (MUX)
A multiplexer selects one of 2n

input data lines and routes it to a single output based on n
select lines. A 4-to-1 MUX has 4 data inputs (D0–D3), 2 select lines (S1S0), and 1 output
(Y).



## Demultiplexer (DEMUX):

A demultiplexer routes a single input to one of 2n output lines based on n select lines. A 1-to-4
DEMUX has 1 data input (D), 2 select lines (S1S0), and 4 outputs (Y0–Y3).




##Discussion:
VHDL (Very High-Speed Integrated Circuit Hardware Description Language) is used to describe and 
design digital circuits in a structured way. In this experiment, combinational circuits such as Multiplexer 
(MUX) and Demultiplexer (DEMUX) are implemented using VHDL code. A MUX selects one input from multiple inputs 
and transfers it to a single output based on select lines, while a DEMUX performs the opposite operation by 
distributing one input signal to one of several outputs. VHDL allows these circuits to be modeled, simulated,
and tested before hardware implementation, making digital system design easier and more reliable.



## Conclusion:

The VHDL implementation of MUX and DEMUX demonstrates the design and working principles of basic combinational circuits.
Through simulation, the correct behavior of input selection and signal distribution was verified. 
This experiment helps in understanding how VHDL can be used for designing and analyzing digital circuits efficiently 
and forms a foundation for developing more complex digital systems.
